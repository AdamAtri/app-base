# app-base
The app-base requires NativeScript cli ver. 6.7.8;
To install NativeScript:
`npm i -g nativescript@6.7.8`
Then follow the instructions that arise from running
`tns doctor`

## Once {N}ativescript is setup
To start the server:
`npm run api-server`

To start the app:
`tns <debug|run> android`
